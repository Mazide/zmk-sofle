# 🎮 Sofle Keyboard Cheatsheet

Полная шпаргалка по всем функциям и слоям клавиатуры.

---

## 📍 Основные слои

- **L_MAC (0)** - Базовый слой для macOS
- **L_VIM_MAC (1)** - Навигационный слой для macOS
- **L_SYM (2)** - Слой символов и программирования
- **L_LIN (3)** - Базовый слой для Linux
- **L_VIM_LIN (4)** - Навигационный слой для Linux
- **L_GAME (5)** - Игровой слой
- **L_ADJUST (6)** - Настройки

## 🧭 Навигационный слой (NAV)

Активируется удержанием нижней клавиши большого пальца левой руки.

### Vim-навигация по тексту

**Левая рука:**
```
Q - ⬆️ Начало документа (Cmd+Up / Ctrl+Home)
W - ➡️ Вперёд по словам (Option+Right / Alt+Right)
E - ➡️ Конец слова (Option+Shift+Right / Alt+Shift+Right)

A - ⬅️ Назад по словам (Option+Left / Alt+Left)
S - ⬅️ Начало строки (Cmd+Left / Home)
D - ➡️ Конец строки (Cmd+Right / End)

G - ⬇️ Конец документа (Cmd+Down / Ctrl+End)
X - 🗑️ Delete
C - ⬇️ Page Down
```

**Правая рука:**
```
H, J, K, L - Vim-стрелки (Left, Down, Up, Right)
U - ⬆️ Page Up

Верхний ряд (3, 4):
- macOS: Скриншот всего экрана / области (Cmd+Shift+3/4)
- Linux: Print Screen / Shift+Print Screen
```

### Медиа и система

**Правая рука - верхний ряд:**
```
I - ⏮️ Previous Track
O - ⏯️ Play/Pause
P - ⏭️ Next Track
[ - 🔅 Brightness Down (позиция справа, второй ряд)
Backspace - 🔆 Brightness Up (крайняя правая, верхний ряд)
```

### Браузер

**Навигация:**
```
T (macOS) - Cmd+[ (назад в браузере)
Y (macOS) - Cmd+] (вперед в браузере)

N (правая) - Ctrl/Cmd+Tab (следующая вкладка)
M (правая) - Ctrl/Cmd+Shift+Tab (предыдущая вкладка)
```

## 💬 Быстрые команды (Комбо)

**Только для Linux слоя!** Эмулируют macOS Cmd-поведение через Ctrl для переиспользования мышечной памяти.

На macOS используйте стандартные Cmd-комбинации - они работают нативно.

**Таймаут: 100ms**

### Работа с текстом (Linux)
```
Alt + C - 📋 Copy (→ Ctrl+C)
Alt + V - 📋 Paste (→ Ctrl+V)
Alt + Z - ↩️ Undo (→ Ctrl+Z)
Alt + X - ↪️ Redo (→ Ctrl+Y)
Alt + A - ✨ Select All (→ Ctrl+A)
Alt + S - 💾 Save (→ Ctrl+S)
Alt + F - 🔍 Find (→ Ctrl+F)
```

### Браузер и приложения (Linux)
```
Alt + T - 🆕 Новая вкладка (→ Ctrl+T)
Alt + W - ❌ Закрыть вкладку (→ Ctrl+W)
Alt + L - 🔗 Фокус на адресной строке (→ Ctrl+L)
Alt + Q - 🚪 Quit приложение (→ Ctrl+Q)
```

**На macOS:** Просто используйте Cmd вместо Alt - это работает нативно!

### Общие комбо (на всех слоях)
```
A+S - Esc
S+D - Ctrl
D+F - Alt
J+K - Right Alt
K+L - Right Ctrl
```

## 🎨 Слой символов и программирования (SYM)

Активируется удержанием нижней клавиши большого пальца правой руки.

### F-клавиши
```
Верхний ряд: F1-F10
Второй ряд (правый край) [ - F12
Backspace позиция - F11
```

### Макросы для программирования

**Левая рука - средний ряд:**
```
A - -> (стрелка)
S - => (fat arrow)
D - != (не равно)
F - == (равно)
```

**Левая рука - нижний ряд:**
```
Z - <= (меньше или равно)
X - >= (больше или равно)
```

### Стандартные символы

**Второй ряд (QWERTY):**
```
Q - ! (восклицательный знак)
W - @ (собака)
E - # (решётка)
R - $ (доллар)
T - % (процент)

Y - ^ (карет)
U - & (амперсанд)
I - * (звёздочка)
O - ( (открывающая скобка)
P - ) (закрывающая скобка)
[ - " (двойная кавычка)
```

**Третий ряд (правая рука):**
```
H - - (минус)
J - = (равно)
K - [ (квадратная скобка)
L - ] (квадратная скобка)
; - \ (обратный слэш)
' - ` (обратная кавычка)
```

**Четвёртый ряд (правая рука):**
```
N - _ (подчёркивание)
M - + (плюс)
, - { (фигурная скобка)
. - } (фигурная скобка)
/ - | (вертикальная черта)
\ - ~ (тильда)
```

## 🎯 Переключение между режимами

Нажмите **одновременно 7, 8, 9** (позиции на правой руке, верхний ряд цифр):

```
macOS → Linux → Game → macOS (циклическое переключение)
```

Каждый режим подсвечивается своим цветом RGB:
- **macOS**: Белый
- **Linux**: Голубой
- **Game**: Тёмно-зелёный

## ⚙️ Слой настроек (ADJUST)

Активируется удержанием крайней правой клавиши в нижнем ряду.

### Bluetooth профили

```
1 - Профиль 1
2 - Профиль 2
3 - Профиль 3
4 - Профиль 4
5 - Профиль 5

9 - BT_CLR (очистить текущий профиль)
0 - BT_CLR_ALL (очистить все профили)
```

### Переключение USB / Bluetooth

```
- (минус) - OUT_USB (переключить на USB)
= (равно) - OUT_BLE (переключить на Bluetooth)
```

### RGB подсветка (underglow)

```
Q - RGB_TOG (вкл/выкл)
W - RGB_BRI (ярче)
E - RGB_BRD (темнее)
R - RGB_ON (включить)
T - RGB_OFF (выключить)

Y - RGB_HUI (оттенок +)
U - RGB_HUD (оттенок -)
I - RGB_SAI (насыщенность +)
O - RGB_SAD (насыщенность -)

7 - RGB_EFF (следующий эффект)
8 - RGB_EFR (предыдущий эффект)
```

### Backlight (подсветка клавиш)

```
A - BL_TOG (вкл/выкл)
S - BL_INC (ярче)
D - BL_DEC (темнее)
F - BL_ON (включить)
G - BL_OFF (выключить)
```

### Система

```
' (апостроф) - soft_off (выключить клавиатуру)
```

## 💡 Советы по использованию

1. **Мышечная память**: Все комбо (Copy, Paste, и т.д.) работают на одних и тех же позициях клавиш на Mac и Linux.

2. **Vim-навигация**: Используйте левую руку для навигации, правую для стрелок - это быстрее, чем тянуться к обычным стрелкам.

3. **Программирование**: На SYM слое все часто используемые операторы (стрелки, сравнения) доступны одним нажатием.

4. **F-клавиши**: Все F1-F12 доступны на SYM слое в верхних двух рядах.

5. **Медиа**: Управление музыкой и яркостью прямо на NAV слое без отрыва рук от основной позиции.

## 🔧 Технические детали

- **Таймауты комбо**: 100ms для Cmd/Ctrl комбо, 50ms для модификаторов
- **Энкодер**: Громкость на базовом слое, скролл на NAV слое
- **Указатель**: Управление мышью через джойстик на NAV слое
